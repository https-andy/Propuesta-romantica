# Propuesta-romantica
Página web interactiva diseñada para hacer una propuesta de una forma creativa y dulce.

## Características
- Interfaz simple y amigable.
- Animaciones con GIFs adorables.
- Botón "No" que se mueve aleatoriamente para evitar ser presionado.
- Página especial al aceptar la propuesta.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
